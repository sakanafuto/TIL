# テスト概要

Flutter においてテストは 3 種類ある。

- Unit test
- Widget test
- integration test


|                    | Unit | Widget | Integration |
| ------------------ | ---- | ------ | ----------- |
| 信頼度             | 低   | 高     | 最高        |
| メンテナンスコスト | 低   | 高     | 最高        |
| 依存度             | 少   | 多     | かなり多い  |
| 実行速度           | 速い | 遅め   | かなり遅い  |

## Unit test

みたまんま。

import 'package:test/test.dart';

void main() {
  test('unit test', (){
    const answer = 1 + 1;
    expect(answer, 2);
  });
}
